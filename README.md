# 🌾 GramSangam – Rural Resource Sharing & Agriculture Assistance Platform

GramSangam is a Django-based platform designed to empower farmers and rural communities by connecting them with agricultural equipment owners, workers, transport providers, and educational resources through a single digital ecosystem.

The vision of GramSangam is to transform villages into self-sustaining digital communities where farmers can easily access resources, services, knowledge, and opportunities.

---

## 🚀 Features

### 🚜 Equipment Rental System
- Browse available agricultural equipment
- Rent tractors, harvesters, rotavators, cultivators, bore machines, and more
- Equipment owners can list and manage their equipment

### 👷 Worker Hiring System
- Find agricultural workers for farming activities
- View worker details and availability
- Send worker hiring requests

### 🚚 Transport Booking System
- Book transport vehicles for crops and farming materials
- Tractor-Trolley, Pickup, Mini Truck and other transport options
- Manage transport booking requests

### 📚 Education Hub
- Farming Guides
- Government Schemes
- Daily Farming Tips
- Educational Videos
- Agriculture Learning Resources

### 🌦 Weather Information
- Real-time weather updates using OpenWeather API
- Temperature, humidity, wind speed and weather conditions
- Helps farmers make informed farming decisions

### 💰 Mandi Price Tracking
- Display mandi prices for various crops
- Supports better crop selling decisions

### 📰 Agriculture News
- Latest agriculture-related information and updates
- Farming awareness and knowledge sharing

### 🔐 User Authentication
- Registration and Login System
- Role-based access for platform users

## 🎯 Problem Statement

Farmers often face challenges in:
- Finding agricultural equipment
- Hiring reliable workers
- Arranging transportation
- Accessing government schemes
- Getting accurate agricultural information

GramSangam aims to solve these problems through a single digital platform.
